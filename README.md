# python-image-degradator

Requirements:

pip install pillow tqdm
pip install pyinstaller

EXE generation:

pyinstaller degradator.py --onefile

##

This is a simple image degradation, made just for fun to exmplore python functionalities.

You will have 4 options:
* Soft
* Medium
* Hard
* Extreme

Choose one of them, that the program will generate a copy of the image and open for preview.

It will only work with JPG tho.
